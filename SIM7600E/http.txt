------------------
-tập lệnh HTTP   -
------------------

1,AT+HTTPINIT
Khởi tạo dịch vụ HTTP server
AT+HTTPINIT

2,AT+HTTPTERM
dừng dịch vụ HTTP server
AT+HTTPTERM

3,AT+HTTPPARA
Cài đặt các thông số HTTP - ký hơn đọc tập lệnh
AT+HTTPPARA=x,y

4,AT+HTTPACTION
Lự chọn phương thức POST hay GET


5, AT+HTTPHEAD
đọc header của server
AT+HTTPHEAD

6, AT+HTTPREAD
đọc nội dung

AT+HTTPREAD=vi tri bat đầu muốn đọc, vị trí cuối muốn đọc
AT+HTTPREAD=0,100

7,AT+HTTPDATA
Ghi nội dung truyền lên server
AT+HTTPDATA=x,y  		- x là độ dài, y là thời gian

8,AT+HTTPSTATUS
Kiểm tra trạng thái
AT+HTTPSTATUS?

9,AT+HTTPSCONT
Lưu nội dung HTTP application

AT+HTTPSCONT
